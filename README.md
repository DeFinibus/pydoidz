PYDOIDZ
========

A quick python + pygame test in the spirit of Space Invaders.

Requirements
============
- Python 3.9+
- pygame 2+

Running the game
================
Install Python 
install Pygame (pip install pygame)
run main.py (python main.py)

LEFT and RIGHT arrow moves the ship
SPACE - FIRE 

Bugs: 
=====
